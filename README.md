## Gestionnaire d’Étudiants

## Description

Mini-application réalisée avec HTML, CSS, JavaScript et jQuery permettant de gérer des étudiants et leurs notes.

## Fonctionnalités

-Ajouter un étudiant
-Modifier une note
-Supprimer un étudiant
-Affichage dynamique des étudiants
-Couleurs conditionnelles selon la note
-Date d’ajout
-Bouton « Tout effacer »
-Sauvegarde automatique avec LocalStorage
-Statistiques (moyenne, max, min, admis)

## Technologies utilisées

-HTML5
-CSS3
-JavaScript
-jQuery
-LocalStorage

## Installation

1. Télécharger ou cloner le projet
2. Ouvrir le dossier du projet
3. Ouvrir index.html dans un navigateur

## Déploiement

Projet déployé via GitHub Pages.

Lien Github Pages: https://khalilrim.github.io/gestionnaire-etudiants-jquery/

## Instructions de test

-Ajouter un étudiant
-Ajouter plusieurs étudiants
-Modifier une note
-Supprimer un étudiant
-Tester les erreurs (nom vide, note invalide)
-Tester le bouton « Tout effacer »
-Rafraîchir la page pour vérifier LocalStorage
-Tester notes 0, 10, 20
